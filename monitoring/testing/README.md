# Testing Scripts

Collection of scripts for testing and validating the deployed infrastructure.

## Scripts

- `quick-test.sh` - Comprehensive health check for all services
- `test-voting-app.sh` - End-to-end voting app test
- `check-ready.sh` - Readiness check script
- `setup.sh` - Interactive setup helper

## Usage

All scripts should be run from the project root directory:
```bash
./testing-scripts/quick-test.sh
```
